# Mesto

**Mesto** — учебный проект Яндекс Практикума: интерактивная страница с профилем
пользователя и галереей карточек, подключённая к серверному API. Приложение
загружает данные пользователя и карточки с сервера; пользователь может
редактировать профиль и аватар, добавлять и удалять собственные карточки,
ставить лайки и открывать изображения в увеличенном виде.

## Ссылка на опубликованный проект

🔗 milanadel.github.io/mesto-ad/

## Функциональность

- загрузка профиля и карточек с API;
- редактирование имени и описания профиля;
- обновление аватара;
- добавление и удаление собственных карточек с подтверждением удаления;
- постановка и снятие лайков с отображением их количества;
- просмотр изображения карточки в полном размере;
- валидация форм с подсказками об ошибках;
- индикация загрузки на кнопках («Сохранение…», «Создание…», «Удаление…»);
- закрытие попапов по кнопке, клику на оверлей и клавише `Esc`.

### Задание по варианту 3 — статистика карточек

По клику на логотип Mesto открывается модальное окно со **свежей статистикой**,
запрашиваемой с сервера в момент открытия:

- всего пользователей в группе;
- всего лайков по всем карточкам;
- максимальное число лайков, собранных одним пользователем;
- чемпион лайков — автор с наибольшим числом лайков;
- список популярных карточек.

## Стек

- HTML5;
- CSS3 и БЭМ;
- JavaScript (ES6+);
- Fetch API;
- Vite;
- GitHub Actions и GitHub Pages.

## Структура

```text
src/blocks/       стили блоков по БЭМ
src/images/       изображения и иконки
src/pages/        основной CSS-файл
src/scripts/      JavaScript-модули
src/vendor/       шрифты и normalize.css
```

## Установка и запуск

```bash
# установка зависимостей
npm install

# локальный сервер разработки (откроется в браузере автоматически)
npm run dev

# сборка проекта в директорию dist
npm run build

# публикация на GitHub Pages
npm run deploy
```
