# Mesto

Mesto — учебный проект Яндекс Практикума. Это интерактивная страница с профилем пользователя и галереей карточек, подключённая к серверному API.

## Автор

Делба Милана, ТРПО23-2.

## О проекте

Приложение загружает данные пользователя и карточки с сервера. Пользователь может редактировать профиль, менять аватар, добавлять новые карточки, удалять свои карточки, ставить лайки и открывать изображения в увеличенном виде.

Проект выполнен по варианту 3. Дополнительная функция варианта — модальное окно со статистикой карточек, которое открывается по клику на логотип Mesto. В статистике отображается общее количество карточек, первая и последняя созданная карточка, общее количество лайков, самая популярная карточка и список карточек с количеством лайков.

## Функциональность

- загрузка профиля и карточек с API;
- редактирование имени и описания профиля;
- обновление аватара;
- добавление новой карточки;
- удаление собственных карточек;
- подтверждение удаления;
- постановка и снятие лайков;
- просмотр изображения карточки;
- статистика карточек по клику на логотип;
- валидация форм;
- закрытие попапов по кнопке, оверлею и клавише `Esc`.
